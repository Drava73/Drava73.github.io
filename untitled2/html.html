    SA-MP 0.2X выпущен :: Опубликовано SA-MP 20 ноября 2008 г.

    Доступно обновление безопасности SA-MP 0.2 (SA-MP 0.2X). . Рекомендуется обновить все проигрыватели SA-MP и владельцев серверов.

    Список серверов SA-MP 0.2.2 будет прекращен. Клиенты SA-MP 0.2.2 не могут подключаться к серверам 0.2X.

    Это обновление включает в себя исправления из версии 0.2.2 R3 клиент / сервер, а также следующее:

    - Исправлена ​​небезопасная обработка личных сообщений.
    - Исправлена ​​проблема фрагментации пакетов / MTU.
    - Различные обновления безопасности для сетевого кода и логики подключения.
    - Обновления безопасности для двоичных файлов клиента.
    - Убрано сжатие из векторов прицеливания для повышения точности.
    - Добавлены секунды в журналы сервера, а также показывает необработанные входящие соединения UDP.
    - OnVehicleMod / OnVehiclePaintjob / OnVehicleRespray включает в себя исходный идентификатор игрока.
    - Добавлен OnPlayerUpdate, обратный вызов, запускаемый для каждого сетевого обновления игрока. Это поможет на стороне сервера античит.
    - Ограничено количество отсканированных фильтров до 16.

    Версия SA-MP, ранее разрабатываемая как 0.2.5, станет SA-MP 0.3, чтобы избежать путаницы.

    Выпущен клиент SA-MP 0.2.2 R3 :: Опубликовано SA-MP 20 июня 2008 г.

    Чтобы решить недавнюю проблему безопасности, мы выпустили клиент SA-MP 0.2.2 R3. . Это необязательное обновление, хотя настоятельно рекомендуется. Клиент SA-MP 0.2.2 R3 работает на всех серверах SA-MP 0.2.2.

    В настоящее время обновления сервера SA-MP 0.2.2 не требуются.

    Помимо обновлений безопасности, клиент 0.2.2 R3 также исправляет некоторые сбои / ошибки предыдущих версий. Следующие обновления были сделаны:

    - Удален весь код SAC. Это исправляет уязвимость безопасности, упомянутую ранее.
    - Исправлена ​​проблема с текстом буфера обмена, не копируемым извне.
    - Исправлено мерцание времени / тумана / освещения при отключении времени.
    - Исправлены некоторые проблемы со звуком автомобиля (тихие автомобили / самолеты / лодки / вертолеты).
    - Исправлена ​​задержка всплывающих текстур / потоков, когда загружалось много моделей.
    - Исправлено несколько вылетов магазина модов автомобилей.
    - Альтернативная сирена для полицейских машин теперь синхронизирована.
    - Код TextDraw полностью переписан. Теперь он работает в интерьерах и будет отключен с помощью табло.
    - Исправлены сбои, вызванные скриптом DestroyVehicle.
    - Импортированы некоторые из 0.2.5 кода для серфинга транспортных средств.
    - Добавлена ​​команда / fpslimit для настройки ограничителя частоты кадров. Допустимые значения / fpslimit составляют 20-100.
